# 🎵 TikTok Clone Project  
## Fullstack Application – Personal Project

### 📌 Mô tả dự án  
Dự án **TikTok Clone** là một nền tảng chia sẻ video ngắn, mô phỏng các tính năng cơ bản của TikTok như đăng tải video, xem video, tương tác (like, comment), và theo dõi người dùng.

---

### 🚀 Tính năng chính
- 🔐 Xác thực bằng JWT  
- 📤 Tải video lên (Cloudinary)  
- ▶️ Trình phát video và điều khiển  
- ❤️ Thả tim, bình luận video  
- 👥 Theo dõi người dùng  
- 🔍 Tìm kiếm video hoặc người dùng  

---

### 🛠️ Công nghệ sử dụng
- **Frontend:** React.js, Tailwind CSS  
- **Backend:** .NET Core Web API  
- **Database:** PostgreSQL  
- **Authentication:** JWT  
- **Realtime Cache:** Redis  
- **Media Storage:** Cloudinary
